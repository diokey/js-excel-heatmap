Welcome to JS Excel Heatmap.

This simple library tries to mimic the excel heatmap feature :

It basically add background color to a data table to give an easy understanding of data displayed in the table.

To contribute to the project

Clone the project.

$ git clone https://github.com/username/js-excel-heatmap.git
$ cd js-excel-heatmap
# make changes
$ git commit -a -m "Your commit"
$ git push

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
